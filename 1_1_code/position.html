The WINDOWTITLE and STATUS filters are not supported when a remote system is specified. The wildcard character (\) is accepted for the */im option only when a filter is applied. Termination of remote processes is always carried out forcefully, regardless of whether the /f option is specified. Supplying a computer name to the hostname filter causes a shutdown and all processes are stopped. You can use tasklist to determine the process ID (PID) for the process to be terminated.
